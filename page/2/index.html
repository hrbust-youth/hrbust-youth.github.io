吴小凡个人简介

发表于 2023-03-12 更新于 2025-03-27

年级：22级硕士研究生
研究方向：自然语言处理、对话系统
研究内容：基于知识图谱面向对话策略的健康推荐
发表文章：无
研究课题介绍：面向糖尿病健康管理的对话系统是一种任务型对话系统，旨在为患者解答日常糖尿病相关问题并提供个性化健康推荐。管道式架构对话系统一般分解为自然语言理解、对话状态跟踪、对话策略以及自然语言生成模块，对话策略主要负责响应方式推断和响应内容生成。目前响应内容生成面临的挑战主要有：生成内容个性化、可靠性和可解释性等问题。本研究主要关注对话策略中的内容响应这一关键环节，把糖尿病健康管理对话系统的内容响应视作对话场景下的健康诊断，主要研究如下：
（1）构建糖尿病健康管理知识图谱“YouthMedKG”。
（2）提出一种基于知识图谱卷积网络的糖尿病知识图谱向量化表征方法。
（3）提出“知识图谱特征增强+对话策略引导”双路融合模型，实现疾病诊断及个性化推荐。

Diachatbot简介

发表于 2022-11-20 更新于 2024-02-28 分类于 糖尿病健康管理对话系统

Diachatbot（糖尿病健康管理对话机器人）项目把糖尿病健康管理对话系统定位于任务型对话系统，为了保证对话系统响应的可控和可解释，典型的任务型对话系统采用 pipeline 体系结构。因此，与本项目相关的研究包括对话理解与状态跟踪、对话策略、知识图谱及推荐、对话生成这四个方面的研究。

Diachat简介及相关信息

发表于 2022-11-20 更新于 2024-03-24 分类于 糖尿病健康管理对话系统

Diachat是为糖尿病健康管理对话系统构建的中文对话数据集，该对话系统为任务导向型对话系统，主要服务于糖尿病患者，为患者解答日常生活中糖尿病相关问题。Diachat数据来源为在线聊天平台中患者与医生的对话信息。Diachat数据集采用基于dialogue act的表示方式进行意图表示并定义了15个act label。并且，Diachat定义了6个domain涵盖语料涉及的领域，分别为：问题（Problem）、饮食（Diet）、行为（Behavior）、运动（Sport）、治疗（Treatment）、基本信息（Profile）。
